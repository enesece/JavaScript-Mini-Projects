# JavaScript-Mini-Projects

A collection of functional projects built with **Vanilla JavaScript**, focusing on core web development concepts, DOM manipulation, and browser APIs without external frameworks.

## Projects

| Project | Description | Key Concepts |
|:---|:---|:---|
| **[LocalStorage Dark Mode](./01-LocalStorage-DarkMode)** | A persistent theme switcher preserving user preferences via browser storage. | `localStorage`, DOM Manipulation, State Management |
| **[LocalStorage Welcome Screen](./02-LocalStorage-Welcome-Screen)** | A personalized greeting app that persists login state, featuring input validation and logout functionality. | `localStorage` (CRUD), Input Validation, Conditional Rendering |
| **[API-LocalStorage-PokeFinder](./03-API-LocalStorage-PokeFinder)** | A dynamic Pokedex application fetching real-time data from PokeAPI with a persistent favorites management system. | Fetch API (Async/Await), `localStorage` (CRUD), DOM Manipulation, CSS Variables |
| **[Basic Calculator](./04-Basic-Calculator)** | A modern, responsive calculator app with UI. Implements arithmetic operations using `eval()` to demonstrate educational logic concepts. | DOM Manipulation, CSS Grid, Event Delegation, `eval()` Function |

## Tech Stack

- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**

# JavaScript-Mini-Projects

A collection of functional projects built with **Vanilla JavaScript**, focusing on core web development concepts, DOM manipulation, and browser APIs without external frameworks.

## Projects

| Project | Description | Key Concepts |
|:---|:---|:---|
| **[LocalStorage Dark Mode](./01-LocalStorage-DarkMode)** | A persistent theme switcher preserving user preferences via browser storage. | `localStorage`, DOM Manipulation, State Management |

## Tech Stack

- **HTML5**
- **CSS3**
